Holiday Card Help

In order to create and send a custom card, do the following:

  1. Create a script file in asset/js/cards (for example: mycard.js)
  2. Customize the script as desired
  3. Open the URL for index.html in your browser. This should be something like:

    https://c9.io/[your username]/KineticJS/workspace/index.html?card=[ script name]

    For example, if your username is "jane" and you created a script named "holiday.js" the URL would be

    https://c9.io/jane/KineticJS/workspace/index.html?card=holiday

  4. You can add "&to=NameOfPerson" at the end of the URL to personalize it

    https://c9.io/jane/KineticJS/workspace/index.html?card=holiday&to=Olivia

  5. Email this URL to anybody and when they open it in their browser, they will see your card
